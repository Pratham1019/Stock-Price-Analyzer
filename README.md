# Stock-Price-Analyzer


We have used a public api which fetches the real time price of a particular indian stock and crypto currency.

---

##  Getting Started

Follow the steps below to set up and run the project locally.

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v14 or higher recommended)
- [npm](https://www.npmjs.com/)
- A code editor with **Live Server** support (e.g., [VS Code](https://code.visualstudio.com/) with the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer))

---

## 🛠️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Server

```bash
npm start
```

The server will start running locally. Keep this terminal open while using the project.

### 4. Open the Project

Open the main HTML file (e.g., `index.html`) using **Live Server** in your code editor.

> In VS Code, right-click the HTML file and select **"Open with Live Server"**.

Your project should now be up and running in the browser! 🎉
